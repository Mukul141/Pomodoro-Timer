# 🍅 Pomodoro Timer

A sleek and efficient productivity tool built with Flutter to help you maintain focus and manage your work sessions effectively using the Pomodoro Technique.

![Flutter Version](https://img.shields.io/badge/Flutter-3.0+-blue.svg)

## ✨ Features

- 🕒 Customizable focus and break timers
- ⏯️ Intuitive controls (start, pause, resume, reset)
- 🔄 Long break after configured focus sessions
- 🔔 Optional sound notifications

## 🚀 Getting Started

### Prerequisites

- Flutter SDK (3.0 or higher)
- Dart SDK
- Android Studio or Xcode

### Installation

1. Clone the repository
```bash
git clone https://github.com/Mukul141/Pomodoro-Timer.git
cd pomodoro_timer
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```

## 📱 Usage

1. Set your preferred focus and break durations
2. Start your focus session
3. Work until the timer ends
4. Take a short break
5. Repeat the cycle
6. Enjoy a long break after completing your sessions
